# Vim

#### Modes
`ESC` - Command  
`i` - Insert  
`v` - Visual

### Tabs
`:tabe FILEPATH` - Open file in new tab  
`:tabn` - Next tab  
`:tabp` - Previous tab

#### Copy & Paste

Enter visual mode and select text.  
`y` - copy selected  
`p` - paste after cursor

#### Save & Exit
`:w` - Save  
`:q` - Quit  
`:wq` - Save & Quit  
`:q!` - Cancel & Quit  

#### Moving
`j` - Down  
`k` - Up  
`h` - Left  
`l` - Right
