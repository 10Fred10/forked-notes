# Shortcuts

`CTRL` + `p` - Search files.  
`CTRL` + `Shift` + `p` - Commands.

`ALT` + `SHIFT` + `Down` - Duplicate line.

# Zen Coding

```html
table.table.small>thead>tr>th*4
```

results in

```html
<table class="table small">
    <thead>
        <tr>
            <th></th>
            <th></th>
            <th></th>
            <th></th>
        </tr>
    </thead>
</table>
```
